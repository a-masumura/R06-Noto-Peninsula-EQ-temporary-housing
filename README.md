# 令和6年能登半島地震 建設型応急仮設住宅GISデータ
## 概要
令和6年能登半島地震に伴い建設中の建設型応急仮設住宅について、自治体の発表資料をもとに位置情報などを付加して、GISデータ化したデータ（GeoJSON形式）を公開しています。  
以下のリンクより、データにアクセス可能です。  
- GeoJSON形式  
[https://github.com/a-masumura/R06-Noto-Peninsula-EQ-temporary-housing/raw/main/令和6年能登半島地震_建設型応急仮設住宅.geojson](https://github.com/a-masumura/R06-Noto-Peninsula-EQ-temporary-housing/raw/main/%E4%BB%A4%E5%92%8C6%E5%B9%B4%E8%83%BD%E7%99%BB%E5%8D%8A%E5%B3%B6%E5%9C%B0%E9%9C%87_%E5%BB%BA%E8%A8%AD%E5%9E%8B%E5%BF%9C%E6%80%A5%E4%BB%AE%E8%A8%AD%E4%BD%8F%E5%AE%85.geojson)

## 属性情報
- **「都道府県」**	String
- **「自治体名」**　String
- **「団地名」**　　String
  - 「団地名」は、各団地の名称で、団地内の看板などに記載されています。
  - 「団地名」については、石川県発表の「着工済み団地一覧」には記載がないため、現地確認、市町のウェブサイト掲載資料で確認できたものを記載しています。
  - 七尾市はウェブサイトに掲載しているPDFに各団地の団地名、建築タイプを掲載しています。こちらも参考にしています。（[石川県七尾市　建設型応急住宅について](https://www.city.nanao.lg.jp/toshikenchiku/oukyuukasetujuutaku.html)）
  - 内灘町はウェブサイトに掲載しているPDFに各団地の位置図、団地名、建築タイプを掲載しています。こちらも参考にしています。（[石川県内灘町　建設型応急住宅の申込受付について](https://www.town.uchinada.lg.jp/site/earthquake/15940.html)）
  - 志賀町はウェブサイトに各団地の位置図、団地名、建築タイプを掲載しています。こちらも参考にしています。（[石川県志賀町　応急仮設住宅の入居申込について](https://www.town.shika.lg.jp/machiseibi/toshikeikaku/kasetsu_moushikomi.html)）
- **「建設地」**　String
- **「工期」**　String
  - 石川県発表の「着工済み団地一覧」では、住所が同じ（建設地が同じ）団地でも、着工日・完成日が異なる場合は、複数行に分かれています。
  - 建設地の名称に「①、②」などと区別されているため、この丸数字を「工期」列に表記しています。①を「1」、②を「2」と表記しています。
- **「住所」**　String
- **「戸数」**　Integer
- **「着工日」**　　String
- **「完成見込み」**　　String
  - 「◯月下旬（YYYY/MM/DD）」などの表記はYYYY年MM月DD日時点の石川県のPDF「着工済み団地一覧」に基づくことを示しています。特に（YYYY/MM/DD）がない場合は「出典」列のデータソースによります。
- **「完成日」**　　String
- **「完成フラグ」**　　String
  - 着工日が明記されたものは「着工」、完成日が明記されたものは「完成」と記載しています。
- **「出典」**　　String
  - GISデータ更新者が、石川県発表の「着工済み団地一覧」で当該団地の記載を最初に確認した際の、「着工済み団地一覧」の日付を記録しています。必ずしも初出とは限りません。
- **「メモ」**　String
  - 新聞記事等による関連情報、GISデータ作成時の注釈などを記載しています。
- **「建築タイプ」**　　String
  - 「建築タイプ」列は、**「プレハブ型（組立ハウス）」「プレハブ型（ユニットハウス）」「トレーラーハウス型」「ムービングハウス型」「木造モバイル建築ユニット型」「まちづくり型（木造長屋型）」「まちづくり型（木造2階建て）」「ふるさと回帰型（木造戸建型）」「その他」** に分類しています。
  - 石川県は、応急仮設住宅（建設型）について、「従来型」（＝プレハブ）、「まちづくり型（熊本モデル）」（＝木造長屋）、「ふるさと回帰型（石川モデル）」（＝木造戸建風）の3タイプを供給すると発表しています。
  - 従来型について、このデータでは「プレハブ型（組立ハウス）」「プレハブ型（ユニットハウス）」「トレーラーハウス型」「ムービングハウス型」「木造モバイル建築ユニット型」に細分化しています。（資料で単にプレハブ造と書かれている場合は単に「プレハブ型」と記載しています。）
  - 建築タイプについては、石川県発表の「着工済み団地一覧」には記載がないため、現地確認、新聞記事、石川県の他のウェブサイト掲載資料、市町のウェブサイト掲載資料で確認できたものを記載しています。
  - 珠洲市はウェブサイトに各団地の現場写真を掲載しています。こちらも参考にしています。（[石川県珠洲市　応急仮設住宅](https://www.city.suzu.lg.jp/life/3/30/174/)）
  - 七尾市はウェブサイトに掲載しているPDFに各団地の団地名、建築タイプを掲載しています。こちらも参考にしています。（[石川県七尾市　建設型応急住宅について](https://www.city.nanao.lg.jp/toshikenchiku/oukyuukasetujuutaku.html)）
  - 羽咋市はウェブサイトに各団地の間取り図を掲載しています。こちらも参考にしています。（[石川県羽咋市　建設型応急住宅(プレハブ住宅)の入居申込みについて](https://www.city.hakui.lg.jp/sumai_kankyou_anzen/sumai/4/15125.html)）
  - 内灘町はウェブサイトに掲載しているPDFに各団地の位置図、団地名、建築タイプを掲載しています。こちらも参考にしています。（[石川県内灘町　建設型応急住宅の申込受付について](https://www.town.uchinada.lg.jp/site/earthquake/15940.html)）
  - 志賀町はウェブサイトに各団地の位置図、団地名、建築タイプを掲載しています。こちらも参考にしています。（[石川県志賀町　応急仮設住宅の入居申込について](https://www.town.shika.lg.jp/machiseibi/toshikeikaku/kasetsu_moushikomi.html)）
  - 2024年5月21日発表の「着工済み団地一覧」からまちづくり型、ふるさと回帰型が表記されるようになりました。珠洲市見附公園の木造2階建ての仮設住宅についてもまちづくり型と表記されているため、これのみ「まちづくり型（木造2階建て）」と表記しています。
